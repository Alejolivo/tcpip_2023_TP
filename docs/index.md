# Smart Green House

## Resumen

```
InfoTics propone una implementación para un invernadero inteligente que incluye la instalación de múltiples sensores inalámbricos dentro del invernadero. Estos sensores medirán variables como temperatura, humedad, intensidad de iluminación y niveles de dióxido de carbono. Los datos recopilados por los sensores serán almacenados en una computadora central.
La instalación de los sensores permitirá a los agricultores monitorear de cerca el entorno del invernadero y obtener información precisa sobre las condiciones de crecimiento de los cultivos. Esto les permitirá tomar decisiones basadas en datos para optimizar la producción agrícola.
La visualización de los datos a través de una plataforma en línea proporcionará una interfaz amigable para que los agricultores puedan acceder y analizar la información recopilada. Podrán obtener información histórica y realizar análisis para mejorar la eficiencia y rentabilidad del invernadero.

```
